Literal blockの練習
========================================

ref: https://www.sphinx-doc.org/ja/master/usage/restructuredtext/basics.html#rst-literal-blocks

::

    >>> 1 + 1
    2

前の段落から続けることもできる::

    >>> print("this is a Literal block")
    this is a Literal block
